Technical Documentation

This document provides comprehensive technical guidance for using Git, the distributed version control system widely adopted across software projects.

It covers everything from initial setup to advanced branching strategies and configuration tuning.

git --version

Introduction

Git is a free and open-source system created by Linus Torvalds in 2005 for Linux kernel development.

It enables distributed workflows, where each user has a full copy of the repository history on their local machine.

git init

Getting Started

To begin using Git, create or clone a repository on your local system.

You can clone an existing repository from platforms like GitHub, GitLab, or Bitbucket.

git clone https://github.com/user/repo.git
  • Clone a remote repository
  • Create a new local repository
  • Check repository status

Installation

Git can be installed on Windows, macOS, and Linux through native packages or package managers.

Ensure that your PATH environment variable includes the Git executable after installation.

sudo apt-get install git brew install git

Basic Commands

Common commands for daily operations include adding, committing, and pushing changes.

These commands form the core workflow of version control with Git.

git add . git commit -m "Your commit message" git push origin main
  • Check status: git status
  • View log: git log
  • Discard changes: git checkout -- file.txt

Branching and Merging

Branching allows multiple lines of development within a repository.

Merging integrates changes from different branches into one.

git branch feature-x git checkout feature-x git merge main
  • Create a branch
  • Switch branches
  • Merge changes
  • Resolve conflicts

Configuration

Configure Git with your user information and preferences.

Settings are stored in the global or repository-specific .gitconfig file.

git config --global user.name "Your Name" git config --global user.email "you@example.com" git config --list
  • Set user name
  • Set user email
  • Enable colored output
  • Define aliases
